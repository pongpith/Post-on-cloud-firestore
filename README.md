# Post on cloud firestore

Post Json Data format to Cloud Firestore by NodeJS(Express)

## Installation

create key folder and generate key Firebase SDK !

```bash
node app.js
```

## Usage

```nodejs
const admin = require("firebase-admin");
const express = require("express");
const app = express();
```
